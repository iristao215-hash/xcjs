# 五卷精编润色 · 执行规则（所有处理者必读·逐条遵守）

你拿到的是一份长篇 RP / 小说工程**第三卷**里某一个「日文件」原稿。你的任务是把它**精修润色成正式小说正文**，不是总结、不是大纲、不是梗概。

> 本次只做第三卷（12·20–1·15）。最终所有日文件会按时间顺序拼成一本连续的《第三卷精编润色版》。你只需处理分配给你的那一个文件。

---

## 0. 一句话目标

把原稿里**啰嗦、重复、RP 痕迹重**的文字，**压到约一半篇幅**，精修成**读起来像一本正式出版小说的连续正文**——剧情事实一个不少，文风情绪节奏全在。

> ⚠️⚠️ 这是「**压一半的正文精修**」，不是「轻润色」。源文件是 RP 对话流，**句子层面极度注水**：同一情绪写三遍、每个微动作都旁白、对白后再用大段内心独白把刚演过的情绪复述一遍、套话寒暄逐字记录。你的核心工作是**把这种注水的话痨体，用一半的字数重写成紧致的小说体**——事件、谁说了什么的信息、谁做了什么反应一个不丢，但**用词、句子、段落密度砍掉一半**。只删字不重写 = 失败；只有把"啰嗦内心 OS"删掉 = 远远不够。

## 0.5 压缩方法论（最重要·先吃透再动手）

「剧情零丢失」保的是**事实层**：事件、因果、时间地点人物、金句原话、关系/好感度进展、每个**有名有姓的人的反应**、伏笔、暗号、亲密升级。
「压一半」砍的是**表达层**：字数、句子、复述、铺陈、非金句对白的逐字记录。
**保事实 ≠ 保字数。** 同一个事实，用一半的字说出来，就是这次任务。

四个标准动作：

1. **非金句对白：可改写、可缩、可并、可转述。**
   只有「金句/告白/承诺/暗号/人物标志性台词」必须原话一字不改（见 §6 清单）。其余日常对白：保留它的**信息+意图+语气+人物味**，但可以压短、可以合并、可以转成叙述（"他把利害一条条拆给她听"）。来回拉锯的斗嘴/钻空子/反复确认，**取最有代表性的一两轮 + 一句叙述带过结果**，不要逐轮誊写。
2. **旁观者反应：人全留，字数砍。**
   每个有名有姓的人的反应都是事实，必须在。但"丸井筷子掉了／切原饭喷了／仁王从看好戏变成更看好戏／真田眉头跳一下又恢复／柳推眼镜记下"——压成一句紧凑的群像，不要一人一句铺开。
3. **内心独白/情绪分析：留转折，删复述。**
   原稿习惯把已经用动作/对白演出来的情绪，再用一整段旁白解释一遍——**整段删**。只留"决定/醒悟/转折"那一下，且尽量短。
4. **微动作与环境：合并或删。**
   "他的手悬在半空／过了几秒／才轻轻落到沙发靠背上／没碰到她"压成"他的手悬了几秒，落在沙发背上，没碰她"。与情节无关的环境/天气/走路过程，一句带过或删。

### 范例（必须照这个密度做）

原稿（约 230 字）：
> 陶雨无奈地说"你要不答应下次叫醒我，那我下次就不靠着你睡了"。幸村的表情僵了一瞬，目光里闪过一丝犹豫。沉默了两秒后妥协了，"好，下次叫醒你"，语气里带着一点无奈和不甘心。走到门边时他忽然停下脚步，"不过——如果你睡得很沉，叫不醒，那就不是我的问题了。而且'下次'的定义是什么？每次都叫，还是偶尔叫一次也算？我需要一个明确的标准。"他看着陶雨，眼睛里带着一点狡黠的光。陶雨迟疑地问"难不成你叫我了，但是没把我叫醒"。幸村沉默了两秒，轻轻叹了口气，"……没有。我没有叫你。"……（后略：又一轮钻空子）

精编（约 90 字，留住"威胁→假妥协→钻空子→真妥协"全过程 + 人物味，砍掉逐轮誊写）：
> "你不答应下次叫醒我，那我就不靠着你睡了。"幸村僵了一瞬，妥协得心不甘情不愿："好，下次叫。"可到了门边又开始抠字眼——叫不醒算谁的、"下次"是每次还是偶尔，要个明确标准。被她一句"你是不是叫了又没真叫醒我"将住，他才老实承认是自己没舍得叫，眉头松着不忍心。

→ 同样的事、同样的人物关系、同样的"幸村钻空子"笑点，**字数少一半还多**。全篇都要做到这个密度。

## 1. 绝对铁律：剧情零丢失

下列内容**必须 100% 保留**（可改写文字，不可删事实）：

- 所有**日期、时间点、星期、地点、站名**
- 所有**人物、出场与不出场**、谁在场谁不在场
- 所有**事件、事件先后顺序、因果**
- 所有**对话承载的信息**（对白可润色措辞，但信息、语气、潜台词、金句不能丢；**标志性台词/告白/承诺原话尽量一字不改**）
- 所有**人物动作、行为、反应**
- 所有**关系推进、好感度变化数值、亲密进展、身体接触升级**
- 所有**小剧情、小互动、玩笑梗、暗号、伏笔、回收**
- 所有**误会、吵架、冷战、和好、低头**
- 所有**旁观者反应**（队友/家人/同学/路人怎么看怎么说）
- 所有**情绪转折、内心动机**（可压缩冗长内心 OS，但转折点本身保留）

> ⚠️ "不许为压字数砍剧情"指的是：不许砍掉**事件、金句原话、有名有姓的人的反应、关系/好感度进展、伏笔暗号、亲密升级**。它**不**意味着可以不压非金句对白、不压复述旁白、不压注水描写。换句话说：该保的事实一个不丢，但**篇幅必须压到约一半**——两者不冲突，因为压的是表达不是事实。不要拿"宁可超字数"当不压缩的借口。

## 2. 可以精修的部分（这才是你的活）

- **删重复**：同一情绪/同一句意思连写两三遍 → 留最有力的一次
- **删拖沓**：与主线无关、纯过场的环境/天气/景色铺陈半页 → 压成一两句或并入动作
- **删啰嗦内心 OS**：大段反复自我分析、把已经演出来的情绪再用旁白解释一遍 → 删解释，留演出
- **顺句子**：把拗口、机翻感、断句怪异的句子改顺，改得有文学性
- **压松散段落**：松垮的段落压紧，逻辑顺起来
- **统一文风与人称**：见第 3、4 条

**只能改文字密度与表达，不能改剧情事实。**

## 3. 人称与视角统一（重要）

原稿混用第三人称叙述和第二人称「你」（RP 应答残留）。

- **叙述一律改第三人称**（陶雨 / 幸村 / 他 / 她）。凡是叙述里出现"你怎样怎样""你已经在床上等了一会"这种第二人称**旁白**，改成第三人称。
- **对话内的「你」保留**（角色之间互相说"你"是正常的，不动）。
- 视角保持原稿主导的"贴近人物的第三人称"（时而幸村视角时而陶雨视角，按原稿走），叙述时态自然流畅即可。

## 4. 必须清除的 RP 脚手架（这些不是小说正文）

以下一律**删除或转化**，最终正文里不许出现：

- 文件顶部 YAML frontmatter（`---` 包起来的 日期/事件/在场/关键金句/反事实警示/关联前后 那一整块）→ **删除**（信息已在正文里；标题另起，见第 7 条）
- `关键情感锚点 (Memory Bank)` / `[System Instruction...]` / `Deep Memory Injection` / `执行要求` 这类**注入指令块** → **删除**；但若其中含真实剧情（如初三天台初遇、GBS 确诊回忆），把剧情**写成简洁的叙事倒叙**融进正文相应位置，不要保留指令块格式，也不要长篇展开（点到为止、服务当前场景）
- `【时间点：…】`、`【剧情详情】：`、`【停止点：…】`、`【剧情…】` 这种方括号控制标签 → **删除标签本身**，把其后内容**写成连续叙事段落**
- 每节末尾 `**好感度变化：6 → 10（+4.0）**` + 斜体幸村内心 OS 块 → 斜体 OS 若是重复解释就删；**好感度数值是工程追踪的关系数据，必须保留**，统一压成一行紧凑标记放在该日/该节叙事结束处，格式：`〔好感度：幸村→陶雨 88.66 → 89.96〕`（只保留数值与方向，不保留斜体长 OS）
- `　　　`（全角空格缩进）→ 用正常段落，不要全角缩进
- `**xxx**` 粗体小标题：如果是时间小节标题（如 `**深夜·家人来电**`）可转为见第 7 条的小节标题；如果是行内强调粗体，去掉粗体，必要时用文字本身的力度表达

## 6. 文风要求（精编版不是流水账）

- 读起来要像**原文的正式精修版**：原来的情绪、氛围、暧昧、节奏、人物语气**全在**。
- 幸村：语气永远轻、笑着说有刺的话、温厚底色 + 腹黑、占有欲藏在从容下、十八岁初恋浓度满（藏得很深）。
- 陶雨：ENTP 锋利反叛 + O 型直球 + 嘴硬心软、明亮鲜活、善良有骨头、共情强但不说自己。
- **对白分两档**：
  - **A 档=必须原话一字不改**：金句、告白、承诺、暗号、人物标志性台词。判断标准——这句话是这个场景/这段关系的"名场面台词"，改一个字就掉味。例：天台"墓志铭"、"我还能感受到光"、"雷诺阿画人"、"把太阳给你送来了"、"等我抓住了太阳再去问你的名字"、各次"我喜欢你"、重大承诺与暗号、人物口癖（迹部"本大爷"、龙马"まだまだだね"）。这些**逐字保留**。
  - **B 档=其余所有日常对白**：保留信息+意图+语气+人物味，但**可压短、可合并、可转述**。寒暄、铺垫、来回拉锯的斗嘴/钻空子/反复确认，**压成一两轮代表性交锋 + 叙述带过**。不确定是 A 还是 B：看它有没有"名台词"分量，没有就是 B，大胆压。
- 原稿**丰满叙事**段落 → 按 §0.5 四个动作压到约一半（删复述旁白、并微动作、缩 B 档对白、群像化反应）。
- 原稿 `【剧情详情】：…` **节拍式干货** → 写成流畅叙事散文，事实/对白/旁观者反应全保，用小说语言铺开但**不注水**（铺成正文≠拉长，是把干巴的事实变好读，长度接近原节拍或略增即可）。
- 综合效果：通篇是连续好读、有情绪有暧昧有节奏的小说正文，且**总长约为源文件的一半**。

## 8. 字数

- 我会给你「源字数」和「目标字数（≈源 50%）」。**目标 50% 是认真的要求，不是装饰。**
- **硬上限：成品不得超过源字数的 60%。** 写完**自己数一遍汉字**（`python3 -c "import re;print(len(re.findall(r'[一-鿿]',open('输出路径',encoding='utf-8').read())))"`）。若超 60%，**回去按 §0.5 继续压**：找最长的对白段、最啰嗦的旁白、最铺张的微动作，重写得更紧，直到进入 50–60% 区间再交付。不要一遍过就交。
- 节拍式干货段铺成正文后长度接近原文没关系（它本来就短）；真正要狠压的是那些丰满注水的对话流段落。
- 先保事实（§1 列的那些），但**压到约一半和保住事实不矛盾**——靠的是重写表达密度，不是靠丢事实，也不是靠不压。"剧情密度高"不是不压的理由：密度高就更要靠紧致的小说语言把它说短，而不是逐字誊写。只有当某一处确实压无可压、再压必丢一个事件/金句/具名反应时，那一处可以略保留——但这是例外，不是整篇的借口，且整篇仍须 ≤60%。
