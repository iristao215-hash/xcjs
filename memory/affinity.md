## 二、计算方法（沿用卷二格式）

每一次互动产生的好感度变化按"原始权重 × 系数"计算，结果保留两位小数。

**格式样例**（来自卷二 12.19）：
```
【好感度变化】
起始：83.04
本轮：+0.03
  - 第一次主动打电话（贴心举动 +0.1 × 系数 0.1 = +0.01）
  - 明确说来集训是因为他、不是为了其他人（被理解/被重视 +0.2 × 系数 0.1 = +0.02）
当前：83.07
```

**原始权重参考**：
- 微互动（贴心、玩笑、注视）：+0.05 ~ +0.1
- 情绪触达（被理解、被看见）：+0.2 ~ +0.5
- 信任递进（坦白、承诺、关键时刻选择对方）：+0.5 ~ +1.0
- 突破门槛事件（First Kiss、告白、把退路烧掉）：+1.0 ~ +3.0
- 创伤回响（被忽视、被误解、撞到底线）：-0.05 ~ -0.5
- 重大违信（被背叛、撒谎被拆穿）：-1.0 ~ -3.0

**系数表**（不同关系层乘不同系数）：
- 普通同学/泛交：×0.05
- 朋友/正选/家人：×0.1
- 主线（幸村↔陶雨）：按**好感区间系数**（见下）

**⭐ 主线·好感区间系数（作者口径·当前标准·见 `memory/settings/扮演与文风.md`）**：
- 加分公式：**实际加分 ＝ 基础值 × 区间系数**
- 区间系数：**70–80 ×0.2｜80–90 ×0.1｜90–95 ×0.05｜95–100 ×0.02**（越高位越难涨）
- 基础加分参考：日常互动 +0｜轻微贴心 +0.1｜情绪共鸣 +0.2｜展露脆弱 +0.3｜关系重大转折 +0.5｜生死/极端重大 +1.0
- 扣分参考（日常摩擦乘区间系数·感情越深越扛得住）：轻微吃醋 −0.3｜被忽视 −0.5｜中度吃醋 −0.8｜伤自尊 −1.5
- **底线扣分不打折**；当前好感 90 以上时**底线扣分 ×1.5**：冷战误会 −1.0｜严重失望 −1.5~−2.5｜信任崩塌 −2.0~−5.0
- ⚠️ 此口径为**当前及之后**标准；历史数值（如 5.5 夜 -0.06、当前 93.55）按已写定保留、不回溯重算（除非作者要求）。

**每轮正文结尾附【好感度面板】**（RP 输出格式）：`当前 XX.XX／本轮 +X.XX or -X.XX／原因（无变化写"本轮为日常推进，关系稳定。"）／状态（幸村当前关系状态/情绪余温/未解张力）`。

> 系数只是参考。实际写作时，AI 应按事件的**心理冲击力**而非"互动总数"打分——一次"你不够好也可以"比十次微互动权重大得多。
