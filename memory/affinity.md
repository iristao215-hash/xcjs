## 二、计算方法（沿用卷二格式）

每一次互动产生的好感度变化按"原始权重 × 系数"计算，结果保留两位小数。

**格式样例**（来自卷二 12.19）：
```
【好感度变化】
起始：83.04
本轮：+0.03
  - 第一次主动打电话（贴心举动 +0.1 × 系数 0.1 = +0.01）
  - 明确说来集训是因为他、不是为了其他人（被理解/被重视 +0.2 × 系数 0.1 = +0.02）
当前：83.07
```

**原始权重参考**：
- 微互动（贴心、玩笑、注视）：+0.05 ~ +0.1
- 情绪触达（被理解、被看见）：+0.2 ~ +0.5
- 信任递进（坦白、承诺、关键时刻选择对方）：+0.5 ~ +1.0
- 突破门槛事件（First Kiss、告白、把退路烧掉）：+1.0 ~ +3.0
- 创伤回响（被忽视、被误解、撞到底线）：-0.05 ~ -0.5
- 重大违信（被背叛、撒谎被拆穿）：-1.0 ~ -3.0

**系数表**（不同关系层乘不同系数）：
- 普通同学/泛交：×0.05
- 朋友/正选/家人：×0.1
- 主线（幸村↔陶雨）：×0.1（基础）·关键节点单算

> 系数只是参考。实际写作时，AI 应该按事件的**心理冲击力**而非"互动总数"打分——一次"你不够好也可以"比十次微互动权重大得多。
