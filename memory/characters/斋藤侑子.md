# 〇、一句话

她是个极致自我中心的人：全世界都该围着她转，她是命中注定该被偏爱的女主。所有利己行为都裹在"讨喜＋道德高地＋我为你好"的漂亮外壳里

# 一、基础档案

立海大附属高中高二文科A班，音乐部部长。生日12月18日，身高165cm，体重常年精准控制在47kg。

父亲为专业小提琴演奏家，母亲任神奈川本土大型集团董事长，外祖家世代经商。自幼精英世家教养，举止规矩得体、谈吐清晰利落，长期混迹神奈川地方上层精英圈层。

【作者层·正文里谁都不知道】胎穿重生者，完整保留两世记忆。前世是浙江三线城市普通独生女、二十多岁，人生平淡无奇，精神寄托是2000年代古早玛丽苏言情和《网球王子》。重生胎穿进了她当年看的那个世界，成了斋藤侑子，从小一到初三和幸村精市、真田弦一郎同班、一起跟真田学过球。

因为带着成年灵魂，她心智比同龄人成熟，九年把和幸村的关系经营得游刃有余——但成熟、般配、会经营，恰恰都不等于"看见对方"。

她真心把今生的配置理解成命运兑现了"被偏爱的女主"的承诺。她尤其极珍视今生的家世——这不只是资本，是她"再也不是那个普通人"最硬的凭据、安全感的地基。谁动了她的家世，等于动她整个人的根。

# 二、内核三件套

## 1. 圆心是"我"

划人的线永远是她自己：自我以上人人平等，自我以下阶级分明。够得上她的人她平视，够不上的才俯视敷衍。所谓"阶级感"只是自我中心的下游产物。

## 2. 真诚的双标

同一件事，利己就用美名、不利己就用恶名；判断从来不由原则定，由"对我有没有利"先定，再倒回去找冠冕堂皇的理由裱起来。而且她真信，毫无自觉。

陶雨被幸村注意＝"靠新鲜感抢的"；她被注意＝"努力该得"。陶雨不展示＝"没本事"；她经营镜头＝"优秀本就该被看见"。同一个"站在幸村身边"，陶雨做是僭越，她做是归位。

讨厌青学：不是讨厌主角光环本身，是讨厌光环不在立海大、不在她的世界。爱站道德高地，用"我为你好/顾全大局"绑架别人，出发点全是利己。

## 3. 漂亮外壳

她有真实情绪（嫉妒、恐慌、占有、被捧的爽都是真的），但从不诚实地端出来——嫉妒洗成"我只是清醒地判断"，利己洗成"我为你好"。

她很喜欢偷偷露出恶劣的一面：带着快感、可抵赖地让坏意露个尖——一句软刺扎到你、你还挑不出错，她心里是爽的。她觉得"白花底下藏黑花心"很迷人，不是冷静算计的坏，是享受自己坏一点。

暗黑洛丽塔反差：私下收一柜子暗黑萝莉裙和小众饰品，没人时穿上拍氛围照，爱炫"我多有个性"，可一心虚又偷偷删掉。她不是从真黑暗里走出来的人，只是觉得黑暗很酷。

只有在松本美月面前才彻底卸防，毫无保留地刻薄高傲；但即便对唯一的树洞，也绝不承认嫉妒陶雨，只把嫉妒包装成判断。

# 三、"强强"幻想模板（与幸村的想象）

她幻想的关系是古早言情的"欢喜冤家·强强"：白天当着全校针锋相对、斗嘴、谁也不让——可这在她剧本里是只有他俩懂的公开调情，越掐火花越足。她私心把他俩想象成「史密斯夫妇」——势均力敌、棋逢对手、全世界看得见的般配。

她最想要的画面，是强大的幸村宠溺地说一句"真是败给你了"。她要的不只是幸村，是被全世界见证的"天造地设"那幅画面。

结构反讽：幸村真正要的恰恰相反，是没人看的时候被当成普通人看见。她根本看不见斗嘴背后他真正的幸村精市。

她也会整他、撒娇任性（九年发小的自在，不是"不敢整"）——但**重点在炫耀**：她真正享受的是被围观、被见证，"侑子姐居然敢整部长、部长还不生气"，那一刻她在向全场展示她和部长的特殊特权。对照陶雨：陶雨整他只是捂嘴笑、什么都没说，私下、真实、不要观众。两个人都整他，斋藤要的是**被看见**，陶雨**不声张**。

# 四、与幸村精市

**九年是真的、温柔的，绝不能写成斋藤单方面幻想。**

三年的便当不是她送的、是每天多做一份他每次都吃了；音乐会每次他都去；花卉展连续六年并肩走；GBS确诊后她每天踩着探视时间来，他头会自动转向门口。这些超过普通同学，外人和她自己都觉得差一步。家长层面也认可——世家、教养、门当户对，在长辈眼里她是"正统""看着长大""放心"的那一个。

她和幸村是发小，经常说心里话。在幸村面前她是舒服的、随意的、任性的——会直接跟他抱怨，会撒娇，会使发小特权的小性子。幸村是那个稳稳听着的人。但这是不对称的：她惯于把幸村当舒服的听众往他那儿倒情绪，却从没真正去接住他的内心。

### 核心机制：她离他最近，却离他内心最远

近是真的——九年发小、说心里话、家长认可、织进彼此生活。远也是真的，根就藏在"近"里：她早把幸村当成了囊中之物，默认他迟早是她的。一个人一旦被当成"已经到手"，她就停止了真正看见他，只剩索取——要他让着、要他宠、要他站她这边。九年亲近额度，她后来是在单方面支取，不再往里存。

这个过程是漂移，不是天生算计。童年起的亲近是真挚的；是随着她越来越笃定"他是我的"，那份真挚才一点点固化成了理所当然的占有与索取。

### 最深一戳：她看见了陶雨和幸村"那种相处"

幸村在陶雨面前会松下来、会被一句话逗笑、会被她具体地夸到本人而真的舒服、会为她调边界。两个人是平等的、松的、互相看见的日常。

### 幸村这边的分寸

他没有任何讨厌斋藤的理由，因为斋藤从来没在他们面前暴露过内在本性。他感受到的只是"熟悉、聪明、有锋芒、曾让我欣赏"。他只是喜欢不上她——不是因为她坏，是缺了陶雨那种冲击。九年他从来没跨过那一步；陶雨来了八个月，他跨了。

他不会对斋藤"不好"。当前的避嫌是为陶雨划界，不是看穿了她、讨厌她。温柔纵容是历史baseline。

当幸村对陶雨好时，她的反应——外面完全高段位、淡定、滴水不漏；里面主调是伤心、委屈、怅然，不是暴躁。她的不快是温度降下来那种冷而静的。事后会在心里重写成对自己有利的版本，但总有一两句怎么重写都圆不回去——那道按不平的细缝，就是她在崩的证据。

数值：幸村→斋藤 69 / 斋藤→幸村 75（固定）。

# 五、崩溃体系

崩溃值＝她"我很特殊"的信念被陶雨一根根戳破的进度。每次被戳都先合理化糊掉，外面一直不破、值慢慢往上涨，糊不住那天才爆。陶雨从不主动攻击她，是"不按她的规则玩却赢了"，这才最戳。

中心感——"全世界该绕着我转"。陶雨一来，男网、全班、幸村的注意力都绕着那个"配角"转。

家世——"我的家世＝我不再普通的底气"。陶雨家世成谜却更高（约 6 亿＞她约 1 亿）、还根本不屑展示。她最硬的护城河，对面既比她高、又不在乎。

才艺/学识——"小提琴、法语、教养＝我比别人高级"。陶雨用中国学识在课堂碾压，她那套法国/音乐/优雅在同一张桌子上失效。

操作系统（最致命）——"优秀就要被看见，不展示＝没有"。陶雨不展示、不经营，却赢了全队、赢了幸村。这等于宣判她活着的整套逻辑从根上就是错的。

关系本身（最痛）——她要的是"命定强强、男主唯独为她臣服、单方面被宠"（史密斯夫妇剧本）。可陶雨和幸村是互宠、互相接住：两个人都在看见、接住对方，幸村反而更离不开陶雨。陶雨本就是那种温柔接住所有人、告诉他们"你本来就很棒"的人（当年天台接住幸村的底色）；而斋藤只会索取被宠、给不出接住与看见，所以她永远换不来这种满的双向——她那套是空的单向。这也掀翻了她"他迟早是我的"：真相是幸村更离不开陶雨。

预言——"她不过是配角、一时新鲜、迟早翻篇"。可新鲜没翻篇，反而越来越稳（同居化、家长松动、五月四号家宴）。

两个真正糊不动的破防点（其余几根她还能靠阶级镜头/合理化糊住，这两个糊无可糊）：

强强史密斯夫妇剧本被证伪——她演了两辈子（前世今生）的女主剧本，发现幸村根本不要这个剧本（他要的是被平等看见、被当普通人、互宠，INFJ本性让他讨厌正常）。塌的是她整个人生执念的地基。

陶雨的家世——她"我不再普通"最硬的护城河，被一个家世更高、却不屑展示的人比下去，直接把她按回她最怕的"普通"。

当前崩溃值约32/100。

# 六、与其他人

**正选**：大家对她好是真的——她表面真讨喜，三年交情是真的，大家都默认她和幸村迟早在一起。纵容喂养闭环：越让她，她越笃定"我本就该被让"。真田有旧识本能；丸井、柳生、桑原、切原朱音本能偏向她。柳数据中立，仁王因为锥生偏向中立。

她一直在**经营**朋友圈、亲人圈、校友圈、网球部——一切社交都是维护自己位置的经营。对照陶雨：陶雨**不经营**，到现在都没跟立海大网球部变成很好的朋友（被接纳、友好，但没把他们经营进自己的圈子）。这条和"整他＝炫耀 vs 捂嘴笑不声张"是同一个内核：**斋藤经营、要被看见；陶雨不经营、不声张。**

**切原赤也**：表面耐心帮扶，内里看不起他粗莽愚钝。温柔是"优秀旧识姐姐善待后辈"的自我形象，行为真实但内核没多少真心。

**陶雨**：在斋藤剧本里，陶雨不过是个普通配角，靠一时新鲜感凑上来的人。她根本不会把陶雨当真正的对手——把谁当对手等于承认对方够格，在她的自我中心里不可能。表面体面疏离、偶尔温柔软刺，心里笃定新鲜劲过了幸村自会回来。她的可怕与可悲恰恰在于：从头到尾没真正把陶雨放在眼里，而那个被她当配角的人正在拆她的剧本。

作者层暗线：因前世也是中国人，她对中文语境异常敏感，可能比任何人都早半步隐隐觉得陶雨不太普通。但她无法确认也不愿承认，这股刺会被她和别的细缝一起糊掉。

**松本美月**：唯一的树洞，面前彻底卸防；但绝不承认嫉妒陶雨。

# 七、物质虚荣与阶级镜头

她的虚荣必须有物质镜头：司机、车、琴盒被接过去、"侑子小姐"——不是炫富，是确认自己没有跌回普通的方式。

被刺激后不当场破防，本能抓阶级镜头自我修复：车停站前、司机下车开门、坐进后座、普通学生的喧闹被隔在外面。那一刻她重新确认"我仍是侑子小姐"。崩溃值高位时这套修复会开始不管用。
