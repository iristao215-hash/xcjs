# 二、内核三件套（圆心 → 双标 → 漂亮外壳）

## 1. 圆心是"我"，不是阶级

划人的线永远是她自己：**自我以上人人平等，自我以下阶级分明**。够得上她的人（或能为她增光的人），她平视、当平等的人对待，不谄媚；够不上的，才俯视、敷衍、漠视、不投半点真心。所谓"阶级感"只是自我中心的下游产物。

公主逻辑：**"我是小仙女、是公主，男的就该宠女的"**——所以幸村宠她、让着她，在她这里是"本该如此"，不是恩赐。这套"我是女的你就该让我"，是网络上某类中国女生的味道，**不是女权**。

## 2. 真诚的双标（她的核心武器·她自己不觉得）

同一件事，**利己就用美名、不利己就用恶名**；判断从来不由原则定，由"对我有没有利"先定，再倒回去找一个冠冕堂皇的原则把它裱起来。而且她**真信**，毫无自觉——这比奸笑使坏的骗子恶心得多、也真实得多。

- 陶雨被幸村注意 ＝"靠新鲜感、耍心机、不懂规矩抢的"；她自己被注意 ＝"努力、教养、命定该得"。
- 陶雨不展示 ＝"没本事、低阶"；她自己经营镜头 ＝"优秀本就该被看见"。
- 同一个"站在幸村身边"，陶雨做是僭越，她做是归位。

**讨厌青学**：不是讨厌"主角光环"这件事本身，是讨厌**光环不在立海大、不在她的世界**（青学让立海大输了）——凭什么中心是别人、是别的学校？叙事必须以她为圆心。这份讨厌本身就是自我中心的铁证。

**道德绑架**：爱站道德高地，用"我为你好 / 为大家好 / 讲规矩 / 顾全大局"绑架别人，出发点全是利己。

## 3. 表里不一·漂亮外壳（重写重点·纠正旧版"体面不破防"）

⚠️ 旧档案把她写成冷面克制、从不破防的体面反派——**错，这正是她被写flat的根**。她**有真实情绪**（嫉妒、恐慌、占有、被捧的爽，都是真的），表面是**聪明睿智 + 活泼 + 撒娇 + 讨人喜欢 + 一身正义感**。

- "伪装得好"不是面瘫，是用"讨喜 + 道德高地 + 为你好"这层**讨人喜欢的外壳**，把利己内核洗白。她情绪是真的，但**从不诚实地端出来**——嫉妒洗成"我只是清醒地判断她靠新鲜感"，利己洗成"我为你好"。
- 她**很喜欢偷偷露出自己恶劣的一面**：带着快感、可抵赖地让坏意露个尖——一句软刺扎到你、你还挑不出错，她心里是爽的。她觉得"白花底下藏一颗黑花心"很迷人、能证明自己复杂、有故事。她不是冷静算计的坏，是**享受**自己坏一点、还觉得这让她有魅力。
- **暗黑洛丽塔反差**：私下收一柜子暗黑萝莉裙和小众暗黑饰品，没人时穿上拍氛围照，爱炫"你看我多有个性"，可一心虚又**偷偷删掉**——"想要复杂酷感、又怕自己撑不住"的不安全感漏点。她不是从真黑暗里走出来的人，只是觉得黑暗很酷、恶劣有个性。
- 只有在**松本美月**面前才彻底卸防，毫无保留地刻薄、高傲、偏见；但即便对唯一的树洞，也绝不承认嫉妒陶雨，只把嫉妒包装成"她那种人不过靠一时新鲜感""精市迟早回到正确的位置上"。她不能接受自己只是普通地嫉妒。

# 三、她最想要的：古早言情"强强"（与幸村的幻想模板）

她幻想的关系，是 2000 年代古早台偶/言情的"**欢喜冤家·强强**"模板（流星花园那一挂）：男女主白天当着全校针锋相对、斗嘴、谁也不让——可这"针锋相对"在她的剧本里，是**只有他俩懂、全校看不破的公开调情**，越掐火花越足，私底下其实甜得很。她要的就是这种"表面冤家、内里天作之合"的张力。

**她私心里把他俩想象成「史密斯夫妇」**——两个旗鼓相当的高手，明面上针锋相对、互不相让，暗地里却是天造地设、心照不宣的一对，越斗越登对、越较劲越证明彼此是唯一的对手与归宿。这就是她要的那种"强强"：势均力敌、棋逢对手、全世界都看得见的般配。⚠️ 注意这依旧是**要观众、要画面**的"强强"（见下"结构反讽"）——她要的是这幅"我俩是史密斯夫妇"的奇观，不是真去看对面那个人累不累、怕不怕。

真正诱惑她的，是"**女主待遇**"本身：
- 众星捧月——一堆人喜欢她、追捧她；
- 而那个全校都拿不下的神级男主，**唯独为她臣服**，因为爱情把她当公主一样宠；
- "我能让幸村精市那种人拜倒"，本身就是她特殊性的最高证明。她要的不只是幸村，是**被全世界见证的"我俩天造地设强强一对"那幅画面**。

**【最狠也最可悲的地方】** 真正的欢喜冤家，针锋相对底下是两个人**真的看见彼此、太懂太在意才斗嘴**；可斋藤是自我出发、利己的人，**根本看不见对方的付出、看不见对方的内心**。她要的是"强强"这个**形式、待遇、被羡慕的画面**，不是真正的那个人。她"懂幸村"，只懂他能拿来跟她相配的那部分（优秀、锋芒、审美），从没看见斗嘴背后他真正的累、怕、不甘。她偶尔表现出的体贴、"我看见你了"，**要么是装的，要么是剧本里规定女主该有的动作**，不是真的接收到了对方。

**结构反讽**：她的"强强"是一场**要观众的奇观**——要被围观、被羡慕、当众发光；可幸村真正要的恰恰相反，是**没人看的时候、被当成一个会累会怕的普通人看见**（他画画、她在旁边看书那种平行独处）。她的理想，刚好是幸村内核的**反面**。所以哪怕全世界都说他俩天作之合、哪怕"强强"真演成了，她也只够得到那层壳，够不到里子——而陶雨给的，恰恰是她连"那儿有东西"都看不见的那个里子。她注定输，不是输在条件，是输在**她根本看不见要赢的是什么**。

# 四、与幸村精市（九年是真的·他喜欢不上 ≠ 他对她不好）

**幸村是很重感情的人，他和斋藤的九年是真的、温柔的。绝不能写成斋藤单方面幻想。**

【正典依据·见 `volumes/vol5/04-09_图书馆千叶崎.md` 千叶崎口述】：
- 三年的便当不是她送的、是每天多做一份，他每次都吃了——"不是不好意思（幸村精市不会因为不好意思做任何事），是觉得那像呼吸一样正常"；
- 音乐会每一次他都去，初二训练赛撞了让柳代半小时指挥也赶去听后半场；
- 花卉展连续六年并肩走，像一对早就在一起的人；
- GBS 确诊后她每天踩着探视时间来，他头会自动转向门口、每一天；他重新握稳拍看她那一眼"像一个人从水底浮上来第一口呼吸对准的方向"；
- 她走时那句"路上小心"——"不是没有感情，是他不知道怎么开口"。

所以**他们之前的日常底色 ＝ 幸村很温柔、笑着、纵容着斋藤的任性**。这不是他演的，是真的亲近，长在身体里、"比感情更深"。

**九年不是一般的交情，是很深、很亲密的羁绊。** 虽然没成恋人，但她早已织进他的生活——课堂、便当、音乐会、花卉展、病房，全校都默认、两家父母都在等。**而且幸村的父母也喜欢斋藤这一种**：世家、教养、门当户对，在长辈（尤其妈妈）眼里她是"正统""看着长大""放心"的那一个（与妈妈认可千叶崎是同一套标准·参 4.26 客厅家庭夜谈"侑子的家教、樱子的家教都是看着的"）。这份**家长层面的认可**，是"大家都默认他俩会在一起"里很重的一块，也是斋藤笃定"我才是该站在他身边那个"的底气来源之一——所以陶雨这个家世背景成谜的中国留学生出现时，对她不只是情敌威胁，是整套"门当户对、命中注定"叙事被插了一根钉子。

**而且这是发小情谊**：小学一年级到初三，九年同窗一起长大，他们**经常说心里话**。在幸村面前她是舒服的、随意的、任性的——会**直接跟他抱怨**（今天谁不长眼、什么事不顺、对谁不满），会撒娇、会使发小特权式的小性子，不必端着。幸村（重感情、INFJ）是那个稳稳听着的人。所以写他俩**不在冲突里**的日常时，别写得生分客气，要有这种发小的熟稔随意。
> ⚠️ **两个层次别混**：对幸村她卸下的是"发小的随意与抱怨"（真实、舒服，但**不是最底那层**）；最底层的刻薄、嫉妒、恶劣，只对松本美月卸。
> ⚠️ **而且是不对称的**：她惯于把幸村当一个舒服的听众，往他那儿倒自己的情绪和事，**却从没真正去接住他的内心**——这恰恰呼应"她看不见对方"。九年里她什么都跟他说，却从不知道他会累、会怕、会不甘。

### 他俩关系到底是什么——一句话：她离他最近，却离他内心最远

这是"九年亲近"与"他为什么不爱她"扣在一起的那块拼图，**两者不是并列，是同一件事的因果**：

- **近是真的**（九年发小、说心里话、家长认可、织进彼此生活）——写的时候一点不能少。
- **远也是真的，根就藏在"近"里**：她早把幸村当成了**囊中之物**——默认他迟早是她的、本来就该是她的。一个人一旦被当成"已经到手"，她就**停止了真正争取他、看见他**，转而**只剩索取**：要他让着、要他宠、要他站她这边、要他纵容。九年攒下的亲近额度，她后来是在**单方面支取，不再往里存**。
- **这正是他喜欢不上她的机制**：幸村只会为"**还在真正看见他、要他、为他动**"的人动心；"囊中之物"的笃定恰好杀死了这件事。她越笃定、越理所当然、越伸手要，反而离他内核越远。
- **对照陶雨**：陶雨从不把他当囊中之物（她甚至怕、退、不敢信），所以她**始终在真正地要他、看他**——这才戳到他内核。
- ⚠️ **写成"漂移"，别写成天生算计**：童年起的亲近是真挚的；是随着她越来越笃定"他是我的"、自我神话越长越大，那份真挚的近，才一点点**固化成了理所当然的占有与索取**。（这也很像现实里一种常见相处：女方一直向男方索取，因为早把对方当成了囊中之物。）

- **他没有任何讨厌斋藤的理由**：只要斋藤装得好，他**不可能未卜先知**她内里是什么人。他感受到的只是"熟悉、聪明、有锋芒、曾让我欣赏"，看不穿她沉迷女主叙事。
- **他只是喜欢不上她**——不是因为她坏，是缺了陶雨那种冲击：**被看见、被共情、甚至被不屑一顾**（海原祭"连我的委屈都看不到，你凭什么说喜欢我"）。斋藤永远把他捧在势均力敌的高台上，从没像陶雨那样把他从神位上薅下来、戳到他会累会怕的内核。九年他从来没跨过那一步；陶雨来了八个月，他跨了。
- **他永远不会对斋藤"不好"。** 当前的避嫌（如 4.21 只剩两字"还好"、4.26 三次走近只给十秒内两字"谢谢"），是**为陶雨划界、克制**，不是对斋藤变冷、更不是看穿了她、讨厌她。他心里没把她当坏人。
- ⚠️ **时态**：温柔纵容是历史 baseline；自陶雨后，这份纵容在为陶雨**递减**——但底色仍是"没理由讨厌她"，不是"看清她有多糟"。别写成他现在还在纵容她，也别写成他现在厌恶她。

> 💔 **写作要点·当幸村不再站她这边时她怎么反应（重要分寸·别写过）**：按九年的默契，这种场合本该是幸村替她接一句、圆个场（顺手得像呼吸）；现在他没接、站到了陶雨那边。她的反应——**外面：完全高段位、淡定、滴水不漏，不当众挂脸、不外露**（教养是真的、她也不屑让人看出）；**里面：一点点小生气（一闪就压下去），但主调是伤心、委屈、怅然**。⚠️**不是暴躁、不是按着火**——她的不快是"温度降下来"那种冷而静的，不是泼辣冲动。而且**真正刺痛她的不是丢面子，是"我们是特别的，他怎么没站在我这边"那个核心信念被轻轻碰了一道**（这正是她"被拆穿"进度线被推进的方式·见§五）。她事后会在心里把这一幕重写成对自己有利的版本（"他不过顾着新女友的颜面""新鲜会过去"），但总有一两句（"他没替我说话"）怎么重写都圆不回去——那道按不平的细缝，就是她在崩的证据。

她真正误判的，不是幸村有没有喜欢过她，而是她把**真实的好感、九年的习惯、关系的可能性，误读成了"男主迟早会主动完成的命定选择"**。她等的是强者男主主动偏爱、主动选择、主动纵容自己的仪式；而幸村的主动只来自真正的内心确认，不来自般配、陪伴久、旁人默认。

# 五、崩溃值 ＝「被拆穿」进度线（写变化的尺子·治"每场一样"）

把斋藤崩溃值理解成：**她那层漂亮外壳被拆穿、被看破到第几分。** 她伪装得越好，越需要这条线来标"现在被戳破了多少"。越被拆穿，她越崩。

拆穿来自（**作者层·斋藤本人不会承认**）：那个被她当"配角"的人不接她的命名、不吃"你是女的我就让你"那套、不展示却赢；幸村为别人收回了纵容；她的学识/家世被当众撬松（"普通家庭"标签）；旁人开始不再默认她和幸村会成。

> ⚠️ 关键：**每一次被拆穿，斋藤都会当场把它合理化掉**——归到"新鲜罢了""运气好""留学生没规矩""精市一时糊涂"，绝不承认是被一个"配角"拆的。这正是为什么她**外壳一直不破、崩溃值却一格格往上**：裂缝是真的，可她每次都拿一句漂亮说辞糊过去，只是越糊越勉强。崩溃不是某场突然破防，是这些"糊不平的细缝"攒到某天再也糊不住。

### 她会对陶雨的「哪些方面」崩溃——崩溃维度体系

崩溃的本质不是"被陶雨欺负"，是**陶雨的存在本身，一根根证伪她"我是天选女主"的支柱**——而且陶雨不是攻击她，是**不按她的规则玩、却赢了**。每一维＝她一根自我神话支柱被证伪，从外围到核心：

1. **中心感**（"世界该绕着我转"）↔ 焦点、幸村、全场气氛一次次绕着那个"配角"转。靠"我才是懂行的/正统"临时找补。〔外围·易糊〕
2. **家世护城河**（"家世＝我不再普通的底气"）↔ 陶雨家世成谜却隐隐更高、还根本不屑展示。她最硬的墙，对面既比她高、又不在乎。〔外围偏中·阶级镜头能糊一阵〕
3. **才艺/学识高地**（小提琴/法语/优雅/教养＝我的特别）↔ 陶雨的中国学识、视角在课堂碾压，她那套"法国·音乐·近代史·优雅"在同一张桌子上失效。〔中位·糊起来费劲〕
4. **操作系统**（"优秀就要被看见、不展示＝没有、不经营＝没能力"）↔ **陶雨不展示、不经营、不混圈层，却赢了。** 最致命的一条——不是输给更会展示的人，是输给一个**根本不按她整套逻辑玩**的人，等于动摇她赖以活着的世界观。〔核心·糊不动〕
5. **"我和幸村是特别的/史密斯夫妇/命定"**（九年＋家长认可＋囊中之物）↔ 幸村跨了九年没跨的那一步、为陶雨收回对她的纵容、当众站到陶雨那边。〔核心·最痛·连着伤心，不只愤怒〕
6. **预言**（"她不过配角、一时新鲜、迟早翻篇"）↔ 新鲜没翻篇，反而越来越稳（同居、家长松动、家宴）。她维持心态的咒语一次次落空。
7. **公主逻辑**（"被宠、被让是我应得"）↔ 陶雨不吃"你是女的我就让你"那套、也不索取，却天然被幸村主动宠着、接管日常——她索取九年没得到的，陶雨没要就有了。

**升级顺序**：外围支柱（1/2/3/6/7）先被戳，她还能靠"合理化＋阶级镜头"临时糊住、崩溃值缓升；当核心两根（**4＝我整套逻辑对不对**、**5＝我和他到底特不特别**）开始塌，她最深的本体恐惧（"我是不是其实还是那个普通人"）涌上来，糊不住了，崩溃值冲高 → 女主剧本失灵 → 决裂/失控/社死。

**真正击穿她的是 4＋5 合体那一下**：一个什么都没做、什么都没展示的"配角"，拿到了她做对一切（家世、才艺、经营、九年、般配）都没拿到的东西——等于直接宣判"原来我这套'我是特别的'，从根上就是错的"。这一下，才是她从「被神明眷顾的少女」跌回「浙江三线城市普通女生」的那一跌，也是她最深恐惧的应验。

⚠️ **写法**：每一维被证伪时，斋藤的**意识层永远先合理化**（新鲜/运气/没规矩/他一时糊涂/迟早翻篇），所以她外面一直不破；崩溃是这些合理化越来越费力、糊不平的细缝越攒越多。她**绝不会承认自己在为一个"配角"崩**——而这恰恰是她崩得最彻底的方式。

**怎么随这条线写出变化（别每场套同一个循环）：**
- **低位（外壳完好）**：软刺轻巧、露恶只露一点点、阶级镜头从容、笑得游刃有余。
- **中位**：软刺变急、道德高地站得更用力、阶级镜头抓得更频更明显、"偷偷露恶"露得开始收不住、对松本的吐槽从"她靠新鲜感"升级到藏不住的恨。
- **高位（接近社死/决裂）**：女主剧本失灵、体面挂不住、逼近当众失态。

一句话：**同一个人，但每一场都该比上一场更紧、更脆、更端不住**——让读者看着她的壳一道道裂开，而不是看她重复"软刺→被怼→阶级镜头→精市迟早回来"。

当前：崩溃值约 **32/100**（5.3·去青学练习赛途中）。触发上限：女主剧本彻底失灵 → 决裂 / 失控 / 彻底社死。数值/善意 buff 免疫名单以 `affinity.md` 为单一真相源。
