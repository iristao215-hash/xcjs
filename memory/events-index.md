## ⚠️ 用法提醒

- **写新场景前**：如果你打算让角色"想起 / 引用 / 回响"过去某事，**先查本索引**，再去对应弧文件读那一段。不要凭记忆引用。
- **写完场景后**：如果引入了新的"专名事件"（场景里给了它名字、未来可能被回忆引用），就在这里追加一行。
- **找不到的事件**：本索引覆盖度不是 100%，遇到"我记得有一次……"的情况，用 grep 在 `volumes/` 全局搜关键词。
