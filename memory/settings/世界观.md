# 世界观与穿越者设定

穿越者框架（仅作者层，不在正文出现）

金手指： 斋藤——世界的善意（她不知情，作用在环境上让所有人不自觉多一层好感，关键时刻天平微倾）。

100点天赋的分配方式：穿越给了两个人各100点。斋藤巴不得丢掉上辈子——把100点分散到了家世、容貌、才华、社交、网球各处，每样都点了一点，拼出了一个"完美的人"（但骨子里那个中国女孩没换掉）。
立海大对斋藤：所有人都把她当自己人（包括朱音），切原除外。

## 阶级体系

阶级体系： 第一档迹部家（旧财阀/世界性财团，另一个世界）→第二档松本家（五大财阀旁系）→第三档斋藤/幸村/千叶崎家（地方名门，幸村家有常驻お手伝いさん）→第四档真田/柳家（传统文化世家）。陶雨家实际在第二档（资产规模和商业影响力不低于松本家），但所有人以为她是普通留学生。
