# 一、归途·七里滨沿海

晚饭后七点，幸村拎着那只深蓝色周末包，送陶雨回七里滨。出门时一家人在玄关送：美咲扒着楼梯转角不肯下来、临别冲陶雨比了个自创的"三根手指"暗号；爸爸靠在客厅门框上没起身，只朝儿子抬了下下巴；妈妈站得比平时近半步，那句"路上小心"尾音是软的，不像四月里硬邦邦撂下就转身，说完还顿了顿补半句"早点回来"。

门在身后合上，外头是五月的夜风，带着海水的凉和潮咸。沿海的路上散步的人散得差不多了，浪声反倒清楚。陶雨抬头看了看天色，说"走回家吧"。幸村没多问，把肩上的包往上托了托，应了声"嗯"——一段路，沿着海走，慢，可今晚他巴不得慢。

他自然地走到她靠海那一侧，替她挡掉夜里翻上来的风。谁也没说话，可今晚这种不说话跟前几天那种不说话是两回事：前几天的沉默里埋着刺，今晚这种，是两个平时都转得太满的脑子，难得一起空下来。走到半路他腾出一只手，扣住她的手指，十指交错——这几天没牵成的，今晚他要补回来。他半是坦白半是恶趣味地说，今天她来敲门那会儿，他其实早就在想晚上怎么找借口去七里滨了。

# 二、"平静"是壳

陶雨笑出声，回他："我敲你门的时候，你一脸平静，我还以为你不想看到我呢。"

幸村没急着辩。他停下脚步，并不掩饰地把那副"平静"拆给她看：那天下午他把跟她的对话框打开过两次，一个字没敲出去又关掉；上午训练心不在焉，被柳看了好几眼；听见楼下对讲机响、他妈那句"上来吧"还没说完，他就知道是她。他开门前在门后站了两秒，专门把脸压下来的——"那不是平静，阿雨。那是我怕一拉开门，就让你看出来：我等你，等了一整天。"

陶雨却揪住他的衬衫领子，半真半假地兴师问罪：你开门一脸平静没表情，我问你还生不生气，你就一句"进"，进来又一句"坐这儿"让我坐你椅子上，最后还得我来道歉，我脾气是真的好。幸村顺着她这股劲往她那边凑，认了她那些一字一顿的"指控"：那几句一个字的回答，是因为他多说一个字就要压不住了；让她坐他椅子、自己站着离远点，是怕一坐近就忘了端着。"那不叫平静，是我用了十八年练出来的那点本事全使上了，才没在你面前散架。"末了还含笑刺一句——你脾气好？好到把一句"我伤心"揣了四天，才肯走到我家门口。

# 三、谁先扛不住·旧账

陶雨冷哼一声白他一眼，反手把旧账翻出来：你嫌我四天才来？那你以前冷战的时候去找我的间隔有多长，你要是觉得我这四天短，咱们也可以拉长到那个长度，你现在就可以回家。

幸村被将得笑出声，认输认得干脆。他替她把账报清：海原祭那回三十七天，再往前一个暑假整整一个月、对话框里一个字没有；她那四天，跟他这两笔比，确实给足了面子，"这账我输，认"。可输完他立刻反手一翻：能冷战三个月那个幸村精市，早被她治得不剩什么了；这四天他什么德行，问仁王（"食堂空气硬得像真田的脸"）、问柳（从他手机收进球包看出他撑不住）都知道；他连四天都熬得那么难看、揣着她那瓶茶五天没敢递——"你真当我舍得，跟你拉长到三十七天？"说着他把周末包往上一托，重新牵起她的手，脚步反而朝七里滨那头迈，把她那句"回家"反扣成"跟我回家"——"我这周末的家在哪儿、包在我肩上，你比我清楚。"

# 四、海边·为什么不来·拆与不拆

陶雨没看他，眼睛往前头黑沉沉的海上去，忽然认真问：你真的就这么生气？既然像你说的这么难受，你都不来找我？

幸村脚步真的停住了，那点你来我往的笑收了回去。他顺着她的视线也转向海，并排站着——有些最难的话不该面对面逼出来。他说：从前跟她冷战，三十七天也好一个暑假也好，卡在那儿的东西很简单，是骄傲，是拉不下脸先开口，那种他后来学会了，把脸拉下来走过去就完了。可这次不是骄傲。这次是她把他说的"我会伤心"一条条拆开、推回他脸上，论证给他看"所以你客观上给了我压力"；那句话是他很私底下、平时不给人看的东西，被当材料用的那一下，他头一回尝到"原来我也会被你这样对待"。所以他不知道怎么来：装没事是骗她也骗自己；走过去说"你那样让我难受"，又成了一场吵架、她再把他拆一遍。那几天他人是想来的——手机开了又关、午休都走到她桌边、那瓶茶天天揣着——可脑子一直没想明白，到了她面前这话怎么说才不会变成第二次。他把最难的也交了出来：还有，他有点怕，怕好不容易把真心递出去再被拆一次。"所以解开这架的是你，阿雨，不是我。我那几天只走到了你桌子边上，叩了两下，没敢再往前走。"

陶雨仍看着海，说出了她自己的那一层："可是我不知道怎么不拆你？怎么才算不拆你？我好像从出生到现在都是这样的思考方式。"

幸村读懂了她那个"不看他"——这话对她太难，难到要找个不必对上眼睛的方向才说得出口。他没去拽她的下巴，顺着她也望着海，并肩着把话一句句往外放：你分得清的。你拆我那场输、告诉我我不是弱是条件压着我；你拆我的右手、一步步推出它不会再走——那些时候你拆的是事，我感激还来不及。五月五号那天你拆的不是事，是我，是我递给你、本没打算拿去争的那一句。所以他从没想让她别拆；他当初看上的就是她这个脑子，她这么想了一辈子不是毛病，是这套东西一路把她保下来的。只有一样：等他哪天不是在讲道理、是在跟她说他自己的时候，她先别急着给他推一个"所以"，那一下先别想着赢——赢不赢得了不要紧，那种时候他根本没在跟她比。末了他反手点醒她：你刚问我"我不知道怎么不拆你"，你知道你那一下在做什么吗？你没在拆我，你把你不会的那块原原本本递到我手上了——你早就会了，只是不知道这就是。

# 五、便利店

到七里滨公寓楼下，陶雨进了便利店买喝的，回头问他要不要买点什么。惨白的灯把两人从海的暗里拽回人间。幸村没往冷柜走，把周末包卸到收银台边，从侧袋掏出一瓶常温无糖茶——她常喝的那种，揣了五天，天天想找时机搁她桌角一直没敢递，"现在不用找时机了，省了我一桩心事"。他扫一眼她拎的几瓶，全是无糖，他早知道。他把东西连茶一起码到台上、掏卡结账，又顺手拿了盒鸡蛋、一盒牛奶——"明早的"，俨然今晚住下、明早给她做饭（这套他现在做得有模有样、不会再忘放盐了）。

陶雨笑着抛下一句意有所指的"那你今天晚上最好不要出来买东西啊"就走。幸村读懂了，几步赶上，从背后贴近、压着声音正擦着她耳后那片最藏不住的皮肤："我本来就没打算再下来……倒是你，自己定了'今晚不许下楼'的规矩，那我可得让你一步都迈不出去。"她却抬头直球把球踢回："我大概知道你想干什么，正因为这样，我建议你好好想想有什么没买的。"幸村从容应对、毫不羞赧，先答她"该有的早买好了，放在你楼上抽屉里、不是今天才放的"；她不信、还点名"你忘了四月二十六那天晚上了？"——他这才明白她说的不是套子，是 4.20 他漏买、4.26 当账翻出来、他认下"好，我来"的那样东西：润滑。他没靠嘴辩，径直回卧室拉开床头柜第二个抽屉，把没拆封的避孕套和那瓶他比对过牌子、为这"下次"备了两周的润滑并排摆出来——言出必行，"放这儿有些日子了"。她惊"你欺负我粗心大意绝对不会看这里"，他坦然认：是我赌你不看，你越不看越说明你信我替你记着；你对自己的东西从来不上心，这些归我管。

# 六、卧室·真正意义上的第一次（第一回）

他俯身停在偏五毫米处，陶雨笑着闭嘴、自己补上了那一吻——在两人的暗语里，谁补上那五毫米谁就先认了。深吻很慢、很深：他含她下唇、舌探进去缠她的舌、舌根相抵地碾，却始终把节奏攥在掌心，渗而不纵。她被亲到腿软，直接坐到床沿，打趣"这可不像慢一点的节奏"。幸村膝盖抵进她腿间圈住她，把"慢"翻成更狠的一面——慢不是轻，是要用一整夜一寸寸来；又替她数那笔她没细记的账："不止四天。上一回碰你是昭和日、那回我连进都没进来只蹭着你；再上一回真正进到你身体里，是四月二十号。你算算我忍了多少天。"

他主导着褪去她的连衣裙（拉链一寸寸拉开、布料堆在腰间又褪净）、自己的衬衫，肌肤相贴；唇从耳后、颈、锁骨一路盖戳似地往下吻，掌心揉她乳房、拇指碾乳尖，她喜欢被这样碰，他早存了档。他俯身用唇舌覆上她的阴蒂、手指探进去勾按她的 G 点，舌与指并用把她送上今夜第一次高潮（与手、嘴的快感同源）。

# 七、第二回·食髓知味

缓了一阵，幸村这个十八岁、常年高强度训练的运动员身体恢复得快；加上久别、又头一回真正尝到滋味，食髓知味，一次远远不够。他读出她已疲惫、体力跟不上、又是头一回，便先疼她——以极轻的吻和不急的手把她重新哄热，"我知道你累……可我还想要你""这一次你什么都不用做，靠着我，剩下的交给我"。
