# 5月8日（金）傍晚·看台长椅

5.8周五图书馆值班结束后，陶雨路过凹下去的男网球场外侧看台，看到场内训练中斋藤侑子以六年旧识身份站在朱音侧后方帮非核心小忙（递毛巾、跟丸井搭话），心里说不清什么感受。她本打算去别的部那边消磨时间，转身时看到千叶崎樱子站在通向校外侧门那段下行台阶上，"早就看见你了"。陶雨走过去坐在她旁边的长椅上。

两人对话从社交客套开始，千叶崎说高三留神奈川、补习班和家里给的路恰好是同一条。陶雨说自己不是这样，不想走父母的路但也不知道想要什么，说有点羡慕千叶崎。这句话超出千叶崎的精算剧本，她慢了半拍，语气出现了整段对话第一道真实的认真。陶雨聊到好奇心，说不想一眼望到头，千叶崎被推到她从不允许自己面对的位置上，说出了"我没好奇过……或者说，我没让自己好奇过"，自己也被这句话吓了一下。陶雨温柔回应偶尔好奇也挺好，千叶崎的手彻底松开，笑出了整段对话里第一个没在演给谁看的笑。

陶雨自然引入青梅竹马的研究话题，然后直接问"学姐你真的不喜欢幸村精市吗"。千叶崎长停顿后承认表层的"喜欢"和"喜欢是被给定的"，但拒绝用陶雨的尺子量她跟幸村的关系。陶雨点破"你看他的眼神太冷静了，跟我跟斋藤都不一样"，千叶崎在三个心跳里把露出的松软收回去，笑出了腹黑遇见识破自己的人的"第三种笑"。她捕捉到陶雨说完后眼神暗下去那一瞬，温柔地问"最近是不是哪一味尝得比平时多一点"。

陶雨坦白自己处于尴尬境地，当初知道千叶崎不喜欢幸村才放心靠近，但现在好像成了插进他们之间的人。千叶崎做了整段对话最深的真情绪溢出："我入的可能不是爱河，是另一条河，比爱河窄、比爱河久、比爱河更没有出口。"陶雨认真接住，提到小时候也跟朋友说过类似的话被说无病呻吟。千叶崎整个人僵了半个心跳，承认"那两个词我也听过，但我连说都没说过，陶同学比我多了一份我没有的勇气"。

对话近一小时结束，两人光明正大从长椅前并肩走下台阶离开。整个画面在网球场上方全场可见，日落把两人轮廓打成金边剪影。场内所有人各按人设接收到这一画面：幸村光线晃眼只看到一半但下一拍偏了五厘米；真田认出来后面无表情、下一拍多了分力；仁王把丸井从经理组桌前拽走保护幸村；柳记三个数据信号后没表情收回；朱音没抬头；佐伯笔停一拍、下意识看斋藤的旧肌肉记忆被自己抓住收回；斋藤从声音里辨认出两双女生鞋的节奏和"两个人一起走"的节拍，抬头看到千叶崎和陶雨并肩，外在零反应但帆布袋提手的手握了一下又松开。幸村在球包拉链中途完整看了一拍，手指在侧袋那瓶无糖茶上按了一下又抽出来，跟真田走向更衣室步幅慢了不止半档。

# 5月8日（金）夜·幸村视角

更衣室里幸村把看到的画面一个个放进去处理。陶雨和千叶崎并肩、距离太自然、走向校外不是男网。他不知道她们何时有了能坐在一条长椅上聊天的交集，千叶崎和陶雨之间唯一的连接点是他自己。他在岔路口停下回头看了一眼七里滨方向，站了三秒转回来上坡回家。晚上在房间里他对着那幅没上色的稻村崎画了十分钟没动笔，打开对话框打了"明"字删掉、又打"今天"两个字全选删掉。九点二十七、九点四十一、十点零三，他翻了一次通讯录停在千叶崎名字上三秒没有点开。第四个没有晚安的夜晚。

# 5月9日（土）上午·训练与独处

周六上午半天训练照常，幸村状态正常，球包侧袋里那瓶无糖茶进入第五天。训练结束仁王问他下午有没有安排，他说"没有"。三人在岔路口分开后他一个人上坡回家，走了五十步回头看了一眼七里滨方向，站了三秒转回来继续走。下午他在房间里待了一整下午，画架前面坐了十分钟没动笔，对话框打开了两次没有发出任何东西。三点十七分他翻到千叶崎名字看了三秒关掉，他在想的不是千叶崎说了什么，是陶雨在找一个不是他的人聊关于他的事。

# 5月9日（土）下午三点·陶雨来了

下午三点陶雨带着果子店和菓子和西友买的草莓出现在幸村家门口。按了门铃，家政接了对讲机认出了她，妈妈在对讲机屏幕前看了三秒后说"请稍等"。铁门打开，陶雨跟家政打招呼后走到门口，笑着对妈妈说"在超市看着草莓比较新鲜想买来送过来"，把草莓和和菓子一起递上。妈妈看到和菓子是虎之屋的包装，接过来交给家政，说了句"上来吧"。美咲从妈妈身后冲出来喊"雨姐姐"。

陶雨在客厅沙发上小声问美咲"我跟你哥哥吵架了，在日本我能去楼上找他吗"。美咲说哥哥在楼上房间里，可以带她去但要先跟妈妈说一声。陶雨让美咲跟妈妈说要上楼看小熊。美咲去告诉妈妈"我想带雨姐姐上楼看小熊"，妈妈看了一眼客厅方向，问美咲房间收拾了没有，美咲说收拾了（其实没有），妈妈说"去吧，别太吵"。

# 美咲房间·出谋划策

美咲房间里，被子团在床角蜡笔散了一桌，Jellycat小熊"小布丁"坐在床头。陶雨告诉美咲逛完房间后可以"一不小心"走到哥哥房间门口，美咲忍笑点头。陶雨坐在地毯上对美咲说了自己的困惑：没想到他这么生气，来了他也不来找自己，但自己也觉得没有做错。美咲告诉她哥哥不是生气的时候那种不理人，他平时生气笑着说完就没事了，这次不笑也不说就自己待着，"我觉得他不是生气，他好像是那种不开心"。陶雨问她隔音怎么样，美咲说关着门听不太清，"但是哥哥耳朵很好，我半夜偷偷下楼拿冰淇淋第二天他就知道了"。

陶雨问美咲"你喜欢哥哥吗"，美咲说可以。陶雨开心地抱上去蹭她，夸她"又好看又聪明的小聪明蛋"，问"我能亲你一下吗"，美咲说可以。陶雨抱了一会儿恋恋不舍地放下，帮她整理头发和衣服，说"可惜下一次就不知道什么时候能再抱抱你了"。美咲说"你快去吧，跟他说说话就好了，哥哥其实很容易心软的，他只是不让别人看出来"。

# 幸村房间·和好

陶雨敲了三下幸村房间的门。门打开，幸村站在门口，白T恤运动裤，头发没打理。表情是一种很安静的东西，没有笑也没有不笑。陶雨笑着来的，看到他的表情收敛了，把手背在身后小心翼翼抬头问"还生气着吗"。他没有回答生不生气，往旁边退了半步说"进来"。

陶雨进了房间，坐在他的椅子上转向书桌面对窗户和大海，支着头看窗外，没有说话。幸村坐在床沿看着她坐在他椅子上看他每天看的那片海。两个人之间隔着一米半的空气和下午的光，安静了很久。

然后陶雨把椅子转过来面对他，拉着他的手，抬头问"你还伤心吗"。她把"生气"换成了"伤心"。幸村的手收紧了，把她的手裹进手掌里。他说"你换了一个字"，拇指碰了碰她的指尖。

陶雨开始认真复盘：这几天问了两三个人，大家聊得都很克制；柳说了一大堆大概意思是她拿他的伤心当了武器；不二说这跟对错没关系跟伤心有关系；所以她想问他还伤不伤心。

幸村开口了。他说那天她把他说的"会伤心会失望会难过"一句句拆开的时候——那些话他对别人从来没说过。她把它们拆开排好告诉他加起来等于什么。逻辑是对的，每一句都是对的，但他没有在跟她讲逻辑。他说"你来了"，然后说"下次直接跟我吵，别去问别人我为什么生气，来问我"。

陶雨摸了摸他的脸，说了三句对不起：对不起我确实不知道会让你这么伤心；对不起我应该早点来找你；对不起我给你说过爱情是不分对错的我还一直纠结对错忘记了你的伤心。

他闭了眼。她说到最后一句的时候他睁开眼，睫毛是湿的。陶雨冲上去抱住了他，没有说话，没有看他，让他靠在她肩上，像以前一样顺着他的半长发。他的身体先僵了一秒然后整个人靠过去，手攥住她衬衫后摆攥得指节发白。他哭得安静，默默掉了几滴泪，眼红了。她在他肩膀上也在默默地哭。

他抬起头叫了五天没叫过的"阿雨"，声音哑了，用拇指擦她脸颊上的泪。她擦了擦他的眼泪，笑着说"你妈妈应该不会突然进来吧，看见我把你惹哭了本来今天突然来就很冒昧了更生我气了"。他抹了一下脸说"没有人惹哭我"，拿纸巾发现她口红蹭在他脸上，说"妈妈进来看到这个比看到哭还难解释"。恶趣味回来了。

> ⭐ 里程碑：陶雨第一次不端任何姿态向幸村道歉，不是因为觉得自己错了，是因为看见了他的伤心。她跨过了"怕变成母亲、怕对人低头"的线。好感度从 93.55 上升至 93.80（+0.25），其中"第一次不端姿态道歉"作为里程碑按全值 +0.20 计入不乘区间系数。

# 厨房大作战

陶雨对幸村说可以做草莓牛奶，然后在小红书上搜到奶盖教程、冰浆做法，迅速进入兴奋的ENTP创作模式。她发现厨房有山药，兴奋地问美咲想不想吃"所有中国小孩都爱吃的"，开始计划蓝莓山药。幸村被她安排去找糖、泡糯米、剥葡萄。她用他手机叫了两单外卖补材料（葡萄、芒果、椰奶、淡奶油、红豆罐头、蓝莓酱、冰块、裱花袋、吸管）。

蒸锅要开火时两人同时卡住：她不会开他家的触控灶台，他在自己家也没用过灶。他去问了家政才把火开上，美咲"哥哥居然会削皮"的震惊和"哥哥居然不会开灶"的喜剧效果并行。三个人一起给糯米饭扇风降温（一把和菓子圆扇、一把草莓盒盖、一把画画硬纸板），扇了一分多钟后陶雨突然意识到可以放冰箱，无语地看着幸村"我没想到你也没想到吗"，他耳朵尖红了说"放冰箱是可以的"但全程保持体面。

陶雨做了三种冰浆（黄瓜、芒果、草莓各两碗，用完所有糯米），蓝莓山药的山药泥装进裱花袋交给幸村"这么需要艺术细胞的活还得你来干"。幸村在白瓷盘上挤出编织纹理、螺旋递减、蓝莓酱对角线穿过白色山药泥，像在画一幅小画。美咲全程屏住呼吸，说"哥哥以后可以去开蛋糕店"。

奶奶中途走过厨房看了一眼，评价"刀工不错"，拿了一颗剥好的葡萄吃了没说好坏。家政看到台面阵势后自动拿出玻璃杯摆好。妈妈从客厅方向看了两秒——三个人挤在台面前、儿子T恤蹭了一块紫色、在自己家厨房里剥葡萄——没有出声。爸爸五点二十回家，走到厨房门口看到三个人背对着他一言不发同步扇糯米饭，笑出了声。他什么都没打断，靠在门框上看完了才说"来了啊"。

家人的反应在同步发生。餐桌上七碗冰浆（含家政一份）和蓝莓山药摆开。奶奶先尝后说"糯米放得好"。爸爸连吃三口说"好吃"，看到蓝莓山药的摆盘问"这是精市弄的"。妈妈最后拿起面前那碗草莓冰浆，第一口小勺，第二口满勺。美咲吃蓝莓山药第一口就喊"好好吃"。家政在桌末端坐着吃完了黄瓜冰浆——她在这个家做了很多年从来是站着的。爸爸对妈妈轻声说"心情挺好的嘛今天""这几天不是一句话不说吗"，妈妈回了三个字"年轻人的事"。

最后一轮是草莓冰奶和葡萄冰奶加海盐奶盖，共七杯。幸村的那杯是草莓少甜的，她单独少放了炼乳。他喝了一口含着吸管多停了一秒。妈妈喝了草莓冰奶后把杯子放下的位置比拿起来时更近了一点。

# 温室·向日葵·朋友圈

厨房交给妈妈和家政做晚饭后，幸村说了一声"走"带她穿过落地窗去了花园温室。温室里向日葵五月四号种下第五天还没发芽，他蹲下碰了碰土确认湿度。陶雨坐在奶奶的旧躺椅上开始发朋友圈，六张照片（葡萄冰茶、草莓奶茶、蓝莓山药、三种冰浆），LINE和微信同发，配文"今天是开奶茶店的陶雨～"。

LINE评论区炸了。丸井问为什么不叫他，切原赤也要喝草莓的。仁王只说"蓝莓山药那张摆法不是你吧"。丸井在第四张角落认出"左边那个是部长的手吗"。照片边缘某处露出一截蓝紫色头发。晓彤一眼看出"这碗不是你家的"，追问在谁家，看到角落后秒懂"行，懂了，当我没问"。柳私聊直接说"碗是幸村家的"。千叶崎十分钟后只点了一个赞，没有评论——她去过那个厨房、用过那些碗，一眼就认出来了。迹部私聊说"第三张不错""摆盘谁教你的"。男网的人从照片角落和摆盘风格里拼出了全貌：他们和好了。

陶雨在评论区发了一条"你们的重点都跑哪去了，没有人夸我吗，除了赤也之外"，引发丸井补救式夸赞和切原"我最先夸的"的争功。她跟迹部私聊约做客吃牛排、问黄金周为什么没约立海大打练习赛。幸村坐在旁边木凳上没看她手机，从她嘴角那个跟迹部斗嘴特有的笑法里认出了她在跟谁聊。他没问。

两人在温室里坐了一会儿。她给他舀了一勺剩的海盐奶盖送到嘴边，他吃了说单吃更好，然后从碗里舀了满满一勺递回她嘴边。她一点没犹豫张嘴吃了。他们用茉莉花茶泡了葡萄冰茶分着喝了同一根吸管。

# 晚饭与回家

六点全家吃咖喱。饭桌上气氛正常，没有人提下午的事也没有人问和好了吗。陶雨的位置在幸村旁边，跟五月四号家宴摆在同一个地方。吃完饭后幸村帮家政收了碗。美咲抱着小布丁来玄关跟陶雨说"下次来还做蓝莓山药"。

幸村从楼上下来时手里多了他每次去七里滨的深蓝色周末包。他对妈妈说"我送阿雨回去，明天训练休息"。妈妈看了他一眼，又看了一眼站在玄关的陶雨，说了三个字"路上小心"。两人从稻村崎一丁目高台往七里滨五丁目走，下坡路灯一盏一盏亮，大约七分钟。
